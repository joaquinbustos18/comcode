Nuestro Conocimiento

HTML

Es el código que se utiliza para estructurar y desplegar una página web y sus contenidos.

CSS

CSS es una función que se agrega a HTML que proporciona tanto a los desarrolladores de sitios Web, así como a los usuarios, más control sobre cómo se muestran las páginas.

JavaScript

JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, ​ basado en prototipos, imperativo, débilmente tipado y dinámico.

SASS

Sass es un metalenguaje de Hojas de Estilo en Cascada . Es un lenguaje de script que es traducido a CSS, SassScript es el lenguaje de script en sí mismo.

REACT

React es una librería Javascript de código abierto diseñada para crear interfaces de usuario con el objetivo de facilitar el desarrollo de aplicaciones en una sola página.

PHP

PHP es un lenguaje de programación interpretado​ del lado del servidor y de uso general que se adapta especialmente al desarrollo web.

MySQL

MySQL es un sistema de administración de bases de datos relacionales
